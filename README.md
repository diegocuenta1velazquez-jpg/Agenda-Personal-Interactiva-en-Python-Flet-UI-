# Agenda-Personal-Interactiva-en-Python-Flet-UI-
Este proyecto es una Agenda Personal Interactiva desarrollada en Python, utilizando la librería Flet para crear una interfaz gráfica moderna, responsiva y fácil de usar. La aplicación permite al usuario crear, visualizar y gestionar notas personales de forma simple y ordenada.

Cómo ejecutar este proyecto

Sigue estos pasos para correr la aplicación en tu computadora usando Python y Visual Studio Code:

1. Clonar el repositorio
git clone https://github.com/tu-usuario/tu-repositorio.git


Luego entra a la carpeta:

cd tu-repositorio

2. Instalar Python

Asegúrate de tener Python 3.10 o superior instalado.
Puedes verificarlo con:

python --version


Si no lo tienes, descárgalo desde:
https://www.python.org/downloads/

3. Crear un entorno virtual (opcional pero recomendado)
python -m venv venv


Activarlo:

Windows

venv\Scripts\activate


Mac / Linux

source venv/bin/activate

4. Instalar dependencias

Este proyecto utiliza Flet, por lo que debes instalarlo:

pip install flet

5. Ejecutar el programa

Simplemente ejecuta:

python app.py


(Usa el nombre real de tu archivo, por ejemplo agenda.py si así lo guardaste.)

Notas importantes

Este proyecto utiliza Flet, una librería que permite crear aplicaciones gráficas modernas con Python.

La aplicación se ejecuta en una ventana de escritorio, no en un navegador.

Si recibes errores al correr el programa, asegúrate de:

Tener la versión correcta de Python.

Haber instalado correctamente Flet.

Ejecutar VSC con el intérprete de Python correcto (ver la esquina inferior derecha de Visual Studio Code).

Puedes modificar el diseño, colores o funcionalidades según tus necesidades.

El proyecto es ideal para estudiantes que desean aprender a crear interfaces gráficas simples con Python.
